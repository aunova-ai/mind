# 🌬️ 마인드웨더(Aunova-Mind) 프로젝트 헌장 (Project Charter)

본 문서는 Amanager(Antigravity)가 정의한 마인드웨더 프로젝트의 목표, 핵심 과제 및 AI 에이전트 군단 구성안입니다.

---

## 💎 1. 프로젝트 비전 및 목표 (Vision & Goals)
> **비전**: "현대인의 마음 날씨를 AI로 진단하고, 초개인화된 정서적 항해 지도를 제공한다."

### 1-1. 핵심 정량 목표 (KPI)
1.  **완독률 90%**: 심리 테스트 시작 후 결과 확인까지의 이탈 최소화.
2.  **바이럴 계수 1.5**: 사용자 1인당 최소 1.5명 이상의 신규 유입 유도.
3.  **커뮤니티 전환율 20%**: 테스트 종료 후 '마음 놀이터' 오픈톡/커뮤니티 가입 비율.
4.  **콘텐츠 대량 생산**: 월 40개 이상의 개성 있는 심리 테스트 자동 생성/배포 파이프라인 구축.
5.  **데이터 자산화**: 10,000명 이상의 익명 마음 데이터 수집을 통한 '대한민국 마음 기상도' 리포트 발행.

---

## 💎 2. 핵심 과제 세부 분췌 (Core Tasks - Zero-Loss)

### 2-1. [데이터] 심리 진단 엔진 고도화
1.  심리학적 근거 기반의 질문 생성 알고리즘 설계.
2.  사용자 답변에 따른 가중치 시스템 및 결과 매칭 로직 구현.
3.  결과 데이터의 Google Sheet 자동 동기화 및 기록.

### 2-2. [마케팅] 심리적 후킹 및 전환 설계
1.  `05_MARKETING_LOGIC`에 따른 '성공 패턴 표준화' 적용.
2.  테스트 결과 공유를 위한 SNS 최적화 비주얼 가이드라인 수립.
3.  바이럴을 유도하는 '개인화된 날씨 메타포' 문구 개발.

### 2-3. [기술] 자동화 파이프라인 구축
1.  n8n/Make를 활용한 질문 생성-배포-수집 시나리오 완성.
2.  Next.js 기반의 경량화된 웹 인터페이스(Vercel) 배포.
3.  DALL-E 3 API 연동을 통한 실시간 결과지 커스텀 이미지 생성.
