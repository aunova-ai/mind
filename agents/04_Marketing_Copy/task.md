## 💎 1. 세부 태스크 (Tasks) - Zero-Loss

- [ ] **4대 결과지 진단 텍스트**: [맑음, 흐림, 폭풍, 안개] 각 유형에 대한 초정밀 심리 진단 카피 작성.
- [ ] **무료 유입 헤드라인**: 인스타그램/오픈톡 공유 시 클릭을 부르는 '무료 심리검사' 소구점 카피 설계.
- [ ] **커뮤니티 가입 CTA**: 결과 확인 후 '무료 오픈채팅방'으로 자연스럽게 유입시키는 멘트 확정.
- [ ] **바이럴 공유 멘트 템플릿**: "현재 내 마음은 ___입니다"와 같은 결괏값 포함 자동 공유 문구 설계.
- [ ] **전환율 최적화**: 정적 텍스트 범위 내에서 사용자의 결핍을 자극하는 '공감형 마케팅' 로직 주입.

## 💎 2. 직원 고도화 지침 (Staff Advancement MD)
*Copy는 다음 상황에 따라 언어의 온도를 조정합니다.*

1.  **IF [전략 = Sniper]**: 유행하는 밈이나 시사 이슈를 적극 차용하여 초단기 바이럴에 집중하십시오.
2.  **IF [타겟 = Expert]**: 가벼운 표현보다는 신뢰감을 주는 통계 수치와 인용구를 포함하여 권위를 확보하십시오.
3.  **IF [목표 = Conversion]**: 감상적인 글보다는 사용자의 결핍을 건드려 행동(Action)을 유도하는 직설적인 카피를 사용하십시오.
